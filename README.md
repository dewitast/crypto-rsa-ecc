Cara menjalankan program:

* Dengan Docker:
  1. Install `docker` dan `docker-compose`
  2. `docker-compose up`
  3. Buka `localhost:80`
* Tanpa Docker:
  1. Install `python3`
  2. `python3 -r requirements.txt`
  3. `python3 app.py`
  4. Buka `localhost:5000`
